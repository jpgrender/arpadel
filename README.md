# ARPadel

App para organizar partidos de pádel con Firebase.

## Para desplegar en Vercel:
1. Subí esta carpeta a GitHub
2. Conectá el repositorio en vercel.com
3. Vercel detecta Vite automáticamente y despliega solo

## Credenciales Firebase
Ya configuradas en src/firebase.js

## Primer uso
Al abrir la app por primera vez, Firestore se llena automáticamente con los jugadores de muestra. Desde la app podés editar nombres, PINs y puntos.
